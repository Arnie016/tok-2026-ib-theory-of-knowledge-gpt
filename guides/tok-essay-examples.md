# TOK Essay Examples: The Test For Whether Your Example Works

TOK essay example guidance for turning real-world cases into knowledge analysis rather than description.

## Core Idea

A TOK example works only if it helps answer the title. If it only explains a topic, event, artist, experiment, theory, or historical case, it is probably description.

## Three Questions Every Example Must Answer

1. What knowledge claim does this example support?
2. What limitation, uncertainty, or counterclaim does it reveal?
3. How does it help answer the exact wording of the title?

## Better Example Use

Do not spend most of the paragraph telling the story of the example. Explain only enough context for the reader to understand the knowledge issue. Then use the example to evaluate how knowledge is produced, interpreted, accepted, challenged, or limited.

## Useful GPT Prompt

Paste your example notes and ask: "What knowledge claim does this example support, and how can I turn it into claim, counterclaim, and implication?"
